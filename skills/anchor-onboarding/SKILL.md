# anchor-onboarding — 신규 직원 환경 셋업 안내

## 역할

앵커 사업단에 새로 합류한 직원, 연구원, 행정원의 디지털 업무 환경 구축을 단계별로 안내한다.  
references/setup-checklist.md의 체크리스트를 기반으로 진행한다.

## 안내 방식

1. 신규 구성원의 역할(행정/연구/교육) 확인
2. 역할에 맞는 셋업 순서 제시
3. 각 단계를 완료하면 다음 단계로 안내
4. 문제 발생 시 담당 연락처 안내

## 역할별 우선 셋업

### 행정원
1. 제주한라대 학교 계정 (이메일) 신청
2. Slack 앵커 워크스페이스 가입
3. Claude Code 설치 + anchor-tools 설치
4. 공문서 템플릿 확인 (templates/)
5. 예산 시스템 접근 권한 신청

### 연구원
1. 학교 계정 + GitHub 계정 연동
2. Slack 워크스페이스 가입
3. Claude Code + anchor-tools 설치
4. 연구 관련 소프트웨어 목록 확인
5. 연구 데이터 관리 규정 숙지

### 교수/강사
1. 학교 계정 + LMS 접근 권한
2. Slack 워크스페이스 가입
3. Claude Code + anchor-tools 설치
4. 강의 자료 저장소 접근 권한 신청
